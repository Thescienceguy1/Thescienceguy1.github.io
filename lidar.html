Project Overview

This project uses an Arduino and a LIDAR sensor mounted on two servos to map an environment in 3D. The device rotates across X and Y axes to scan distances and visualize the space in Processing.

System Components

  • Arduino Uno
  • Two servos for horizontal and vertical rotation
  • LIDAR-Lite sensor
  • Custom 3D-printed mount
  • Processing script for visualization

Process

The scanner sweeps horizontally across 180 degrees and vertically 1 degree per step. Each reading provides X, Y, and Z coordinates based on angle and distance. These points are plotted in real-time using Processing.

Applications

  • Room mapping and spatial awareness
  • Intro to robotics and sensor fusion
  • Potential for integration with SLAM systems
