Mission Overview

The Crater Operation Resource Detection (C.O.R.D.) project is a dual-rover concept featuring a primary “mother” rover and a smaller “baby” rover equipped with snake-like actuators. The mission is to explore the Nobile Crater on the Moon, locate and analyze lunar ice, and support future lunar operations through resource detection and autonomous navigation.

Project Goals

  • Navigate and analyze lunar terrain at Nobile Crater
  • Locate and study lunar ice for in-situ resource utilization
  • Assist in future lunar exploration and astronaut training
  • Break down lunar ice into usable elements using on-board tools

Software Systems

The rover software is developed in C with modular architecture, allowing updates and flexible functionality. Key features include:

  • Obstacle-avoidance algorithms
  • Sensor data processing (temperature and environmental)
  • Real-time monitoring and diagnostics
  • Machine learning for sample analysis

Hardware Design

  • “Gear-like” wheels for traction on rough terrain
  • Snake actuators for access to tight spaces
  • Rock Abrasion Tool (RAT) to expose lunar ice
  • Mass spectrometer to analyze ice samples
  • 1 km tether to power the baby rover

Power System

  • Gallium Arsenide solar panel for high-efficiency output
  • Dual Li-ion batteries (1.34 kWh capacity) for sustained operations
  • Power Management via Clyde Space’s Starbuck Micro PMAD
  • Hibernate mode over lunar nights to preserve power

Transportation to Lunar Surface

The rover will be delivered via CLPS using Astrobotic Technology’s lander, selected for its precision landing capabilities and proven lunar delivery experience.

Deployment & Navigation

  • Mother rover deploys first and stabilizes
  • Baby rover performs diagnostics, then begins crater exploration
  • Live data feeds support real-time adjustments and pathfinding

Future Enhancements

  • Advanced autonomy for both rovers
  • Multi-functional snake bot improvements
  • Enhanced environmental and sample analysis tools
